# RadomPassW
RadomPassW is a simple, functional, and reliable random password generator in Python, coded in under 10 minutes. It includes uppercase letters, lowercase letters, digits, and special characters for strong passwords


If Type = 'r', you get a random *i_securyti* character password containing any type of character for maximum security.

If Type = 'p', then you get a passphrase of *i_securyti* words.
